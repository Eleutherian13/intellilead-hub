# Server

Backend server for IITR Productathon application.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file based on `.env.example`:

```bash
cp .env.example .env
```

3. Start the development server:

```bash
npm run dev
```

## API Endpoints

- `GET /api` - Test endpoint to verify server is running
